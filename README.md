# 📝 Laravel Todo CRUD App

## 📌 Description
This project is a web application built using the Laravel Framework that demonstrates the MVC (Model-View-Controller) architecture. It allows users to Create, Read, Update, and Delete (CRUD) tasks stored in a database.

---

## 🚀 Features
- ➕ Create new todo
- 📋 View all todos
- ✏️ Edit existing todo
- 🗑 Delete todo
- 🔍 Search functionality
- ⚠️ Form validation
- ℹ️ About Us page (group members)

---

## 🛠 Technologies Used
- Laravel 10/11
- PHP
- MySQL
- Tailwind CSS
