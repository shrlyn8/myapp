Todo Web App Using Laravel

Description:
This project is a web application built using the Laravel Framework that demonstrates the MVC (Model-View-Controller) architecture. It allows users to Create, Read, Update, and Delete (CRUD) tasks stored in a database.

Features: 
1. Create new todo
2. View all todos
3. Edit existing todo
4. Delete todo
5. Form Validation (when deleting)
6. Search functionality
7. About Us page (group members)

Technologies Used:
Laravel 10/11
PHP
MySQL
Tailwind CSS

INSTALLATION
1. Clone repo: git clone https://github.com/shrlyn8/myapp.git
2. Open project: cd myapp
3. Install dependencies: composer install
4. Setup environment: cp .env.example .env
5. Generate key: php artisan key:generate
6. Configure database in .env
7. Run migrations: php artisan migrate
8. Start server: php artisan serve

USAGE
Open browser: http://127.0.0.1:8000/todos
