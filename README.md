# 📝 Laravel Todo CRUD App

## 📌 Description
This project is a web application built using the Laravel Framework that demonstrates the MVC (Model-View-Controller) architecture. It allows users to Create, Read, Update, and Delete (CRUD) tasks stored in a database.

---

## 🚀 Features
- ➕ Create new todo
- 📋 View all todos
- ✏️ Edit existing todo
- 🗑 Delete todo
- 🔍 Search functionality
- ⚠️ Form validation
- ℹ️ About Us page (group members)

---

## 🛠 Technologies Used
- Laravel 10/11
- PHP
- MySQL
- Tailwind CSS

---

## ⚙️ Installation

1. Clone repo:
git clone https://github.com/shrlyn8/myapp.git

2. Open project:
cd myapp

3. Install dependencies:
composer install

4. Setup environment:
cp .env.example .env

5. Generate key:
php artisan key:generate

6. Configure database in .env

7. Run migrations:
php artisan migrate

8. Start server:
php artisan serve

---

## 🌐 Usage
Open browser:
http://127.0.0.1:8000/todos
